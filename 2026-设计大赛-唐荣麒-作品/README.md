# 🐎 小马接金币游戏

> 2026碧海宝安区设计大赛作品 - 作者：唐荣麒 ©2026

## 🎮 游戏介绍
一款使用 HTML5 Canvas + JavaScript 开发的网页游戏，控制小马接住从天而降的金币，收集道具，触发超级金币雨！

## 🚀 在线游玩
直接访问：[https://你的用户名.github.io/小马接金币游戏/](https://你的用户名.github.io/小马接金币游戏/)

## 🎯 游戏特色
- 完全在浏览器中运行，无需下载安装
- 精美的游戏画面和动画效果
- 三种道具系统：密度+、金币雨、时间+
- 多种金币类型：普通、特殊、超级
- 完整的计分系统和时间管理

## ⌨️ 控制说明
- **← →**：移动小马
- **1**：使用密度道具
- **2**：触发金币雨
- **3**：使用时间道具
- **R**：重新开始游戏
- **P**：暂停/继续
- **ESC**：退出游戏
